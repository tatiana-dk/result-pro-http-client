# QuickRequest — запросы быстро, просто, без лишнего кода
Новая HTTP библиотека

**Главная задача:** Библиотека решает задачу упрощения и стандартизации HTTP-запросов в браузере или Node.js с использованием нативного fetch, без лишних зависимостей. Она делает запросы более удобными, надежными и читаемыми, минимизируя boilerplate-код (повторяющийся шаблонный код).

**Полезность:** Полезна для веб-разработчиков, кто хочет легкую альтернативу библиотекам вроде axios или superagent. Она ускоряет разработку API-интеграций, снижает ошибки в обработке ответов, добавляет удобства вроде автоматической обработки JSON или ошибок, и легко интегрируется в проекты на JavaScript/TypeScript. Идеально для небольших приложений, где не нужны тяжелые фреймворки.

## Список фич для первой версии (v1)

Базовая функция для GET/POST/PUT/DELETE с параметрами (headers, query, body).
Автоматическая обработка JSON в запросах и ответах.
Обработка ошибок: кастомные исключения для HTTP-статусов (например, 4xx/5xx).
Поддержка таймаутов для запросов.
Отмена запросов с помощью AbortController.
Простая конфигурация по умолчанию (baseURL, default headers).

## 2–3 идеи на потом

Interceptors: хуки для модификации запросов/ответов перед отправкой/после получения (например, для аутентификации).
Кэширование ответов: опциональное хранение результатов для повторных запросов.
Поддержка форм-данных и файлов: удобная отправка multipart/form-data.
